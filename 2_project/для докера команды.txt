Перейдите в папку с вашим проектом, используя команду cd путь_к_папке.

Запустите сборку образа, введя команду docker build -t dockerfile .

Запустите контейнер на основе вашего образа:
docker run -d -p 8080:80 dockerfile


http://localhost:8080 (для проверки в браузере)

docker ps (просмотр списка контейнеров)
docker stop <CONTAINER_ID> (остановить контейнер)
